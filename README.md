# GE_RD_simple

**Install**

```bash
git clone https://github.com/rafadls/GE_RD_simple
cd GE_RD_simple
```

**Requirements**

```bash
pip install -r requirements.txt
```

**Variable**

To set the grid edit the "parameters/listaDeParametros.json" file

**Run**

```bash
cd src/
python ponyge.py --variable
```
